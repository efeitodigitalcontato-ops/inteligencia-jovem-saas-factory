Configuração Necessária: Chave de API do Google Gemini

Você ainda não configurou sua chave do Gemini. Sem ela, a geração automática de posts com Inteligência Artificial para os seus blogs não funcionará. Clique aqui para configurar agora.

Passo a Passo Rápido:

  1. Acesse o site Google AI Studio.
  2. Faça login com sua conta do Gmail (Google).
  3. Clique no botão azul "Get API key" no menu esquerdo superior.

  5. Clique em "Create API key in new project" (ou selecione um projeto existente).
  6. Copie a chave gerada (ela começa com AIzaSy...).
